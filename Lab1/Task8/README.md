Задание 8: Форматированный вывод
Программа принимает на вход строку - разделитель, вам необходимо распечатать числа от 1 до 5, выводя между ними введенный разделитель

Примеры

Входные данные
!
Выходные данные
1!2!3!4!5

Входные данные
$
Выходные данные
1$2$3$4$5

Входные данные
BoB
Выходные данные
1BoB2BoB3BoB4BoB5
