Задание 6: Напишите программу, которая вычисляет среднее арифметическое четырех введенных целых чисел.

Обратите внимание, что числа вводятся в одну строку через пробел

Примеры

Входные данные
2 3 4 5
Выходные данные:
3.5

Входные данные
3 3 4 4
Выходные данные
3.5

Входные данные
5 3 4 5
Выходные данные
4.25
